### Installations
#### Node
```
git clone git@github.com:arrking/node.git
cd node 
git checkout v0.11.16
./configure 
make && sudo make install
```
Verify version of node and nom
```
npm -v # 2.3.0
node -v # v0.11.16
```

#### pm2
```
sudo npm install pm2@0.12.10 -g
```

#### Redis server v=2.8.17
